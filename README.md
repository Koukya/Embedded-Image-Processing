# Embedded-Image-Processing
# 一、說明
本次專題主要透過OpenCV加上OCR(光學字元辨識)來實現車牌辨識功能
# 二、需求
### 功能
1.辨識圖片中的車牌號碼  
2.將入場的車牌紀錄及離場的車牌刪除  
3.能隨時檢視停車場內的車牌  
### 畫面大小
#### 輸入
1080 X 1980 pixels  
29.99FPS  
#### 輸出
720 X 1280 pixels  
# 三、效能
每次一張圖片，30FPS取五張圖片
# 四、介面
1.顯示目前車牌圖片及號碼
2.顯示目前車輛總數及剩餘位置
# 五、限制
1.一次一張車牌
2.中華民國車牌
3.正面車牌(不用正前方)
